# DAML By Example

Learn DAML smart contracts through practical, hands-on examples. A developer-focused guide to building on the Canton Network.

Built with Next.js, Tailwind, and MDX.

## Contribute

All content lives in the `content/` folder. Each subdirectory is a category (e.g. `basics`, `contracts`, `finance`). Each file is an MDX page.

To add or edit content:
- **fork** the repo, add/edit `.mdx` files, and open a **pull request**
- Put frontmatter (`title`, `description`) at the top of each file

The site auto-discovers new categories and files at build time.

```bash
npm run dev
```
